# 高级技巧

这里是一些“官方不一定做复杂功能，但你可以用小技巧实现”的玩法。

## 如何设置打折 / 满减优惠？

为了保持记账逻辑简单，系统暂无复杂的优惠券功能。

**小技巧：**  
你可以创建一个名为“优惠/抹零”的商品，价格设为 **-5 元**。  
结算时把这个商品加入订单，总价就会自动减去 5 元。

## 支持“捆绑销售”或“套装”吗？

建议直接录入一个名为“XX套装”的新商品，并设置好打包价。  
这样在统计销量时也能清楚地知道卖出了多少个套装。

## 我该如何处理售罄的商品？

在库存不足或售罄时，顾客端会自动将该商品置灰并禁止下单。  
你无需手动隐藏或下架商品。
