# 网络连接常见问题

## 手机扫码后显示"连接超时"或网页无法打开？

这是最常见的问题，请按顺序排查：

1. **检查热点**：确保主机（电脑/平板）和扫码的手机连的是同一个 Wi-Fi/热点。
2. **防火墙拦截**：Windows 主机请检查防火墙是否允许了"摊盒"通过，或尝试暂时关闭防火墙。
3. **IP变动**：如果重启过热点，IP 地址可能会变，请在主机端点击"重新生成二维码"。

## 没有网络/信号差能用吗？

**完全可以。** 摊盒是"离线优先"的设计。推荐用一台设备开启**手机热点**组建局域网，其他设备连接它即可，不需要连接互联网，也不消耗流量。

## 我是校园网/公共 Wi-Fi，设备连不上？

公共网络通常开启了"AP 隔离"功能，禁止设备间互相访问。**请务必使用热点**来组建网络，这是漫展现场最稳妥的方案。

## 中途断网了，数据会丢吗？

**不会。** 只要主机端（App端）没有关闭，数据就一直存在。重新连接网络后，手机端刷新页面即可恢复之前的状态。
