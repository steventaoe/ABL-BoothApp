# 突发状况

别慌，现场出摊总会有手滑/下错单/设备没电这种事——大多数都能“可逆”。

## 顾客下错单 / 我想取消订单怎么办？

在摊主配货页面：

- **不要点“完成”**
- 直接点击红色的 **“取消”**

该订单会被废弃，锁定的库存会自动返还。

## 手滑误点了“完成”，想反悔怎么办？

别慌：

1. 去主机的“订单管理”页面  
2. 找到那笔误操作的订单  
3. 把状态修改为 **“已取消”**

系统会：

- 把库存加回去  
- 修正销售额统计

## 主机设备突然没电 / 死机了怎么办？

摊盒采用 SQLite **实时落盘**存储。  
重启设备和软件后，所有商品信息、历史订单和库存数据都会 **自动恢复** 到死机前的那一刻。

:::tip 现场建议
主机端最好配一个移动电源/插线板。不是因为摊盒不稳，而是现场供电就是会不靠谱。
:::
