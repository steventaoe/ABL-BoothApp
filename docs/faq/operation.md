# 现场运营

这一页主要回答：收款码怎么放、订单怎么结、提示音、多设备协同这些“现场会遇到的事”。

## 怎么设置收款码？支持多渠道吗？

在“展会管理”点击“编辑”，可以上传图片。  
如果你同时支持微信/支付宝，可以把两个二维码 **拼成一张长图** 再上传（最省事）。

## 顾客付款后，系统会自动完成订单吗？

**不会。**  
为了保护隐私、也为了规避金融风险，摊盒不接入支付接口。

你听到“支付宝/微信到账”的提示后，需要在摊主手机上手动点 **“完成订单”** 来扣减库存。

## 有新订单时会有声音提示吗？

**有的。** 摊主手机端在收到新下单请求时会播放提示音。

如果你没听到，优先检查：

- 手机是否处于 **静音/勿扰**
- **媒体音量** 是否太小

## 可以多人 / 多设备同时摆摊吗？

支持！只要连入同一个热点，你可以：

- 放 2 台平板作为“点单机”
- 让 3 位摊主都拿着手机作为“配货机”

所有数据会实时同步。
