# 硬件建议

一句话：摊盒对性能要求不高，但对“持续在线”要求高，所以主机的选择很关键。

## 我可以在手机上运行主机端吗？

技术上可以，但 **强烈不推荐**。

原因很现实：

- 锁屏/后台容易杀进程  
- 发热会降频甚至断联  
- 现场你还得拿它收款/沟通，很容易误操作

**推荐用笔记本电脑或平板作为主机。**

## 对设备性能有要求吗？

要求很低：

- 任何能运行最新版 Chrome/Edge 的 Windows 电脑
- 或 Android 8.0+ 的平板/手机

都可以流畅运行。老旧设备当“顾客点单机”也是很好的废物利用。

## 需要一直亮屏吗？

- 主机端建议保持亮屏，或在电源设置里设为 **不休眠**
- 顾客端平板建议设置 **屏幕常亮**，避免顾客点单时突然黑屏
