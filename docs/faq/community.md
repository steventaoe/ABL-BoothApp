# 社区与开源

这里主要是：下载渠道、更新方式、反馈入口、以及开源相关的常见疑问。

## 去哪里下载最新版本？

请认准唯一的官方渠道：**GitHub Releases** 页面。  
任何要求付费下载或“代部署收费”的渠道均为诈骗，请勿上当。

## 我能自动更新吗？

请点击侧边栏的“检查更新”按钮，系统会自动检测最新版本并引导你下载安装包。  
当前版本不支持一键自动更新，需要手动下载安装包进行覆盖安装。

## 遇到 Bug 或有建议去哪里反馈？

- 程序报错：强烈建议在 GitHub 仓库提交 **Issue**（方便开发者追踪修复）
- 使用疑惑/闲聊：欢迎加入 **用户交流群**

## 我是开发者，可以为项目贡献代码吗？

**非常欢迎！**  
无论是修复一个小 Bug，还是开发一个新功能模块，你的贡献都会帮助到更多摊主。  
欢迎提交 **Pull Request**。

## 我能修改软件并重新发布（甚至售卖）吗？

本项目遵循 **MIT 协议**，你确实拥有修改和分发的自由。  
但作为一款旨在“降低同人出摊门槛”的免费工具，我们 **强烈不推荐** 将其包装为商业软件售卖。

请保留原作者信息，尊重开源精神。
