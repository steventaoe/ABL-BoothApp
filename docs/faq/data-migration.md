# 数据安全与迁移

这一页关心三件事：数据在哪里、怎么换机、导出 Excel 有啥。

## 我的数据存储在哪里？安全吗？

所有数据（图片、账本、库存）均存储在你 **主机设备的本地数据库** 中。  
我们没有任何云端服务器，你的营业额数据除了你自己没人知道。

## 换了新设备，怎么迁移数据？

在“全局商品库”页面，点击 **“导出数据包”**，会将所有商品信息和图片打包下载。  
把这个文件发到新设备，在设置页选择 **“导入数据”** 即可。

## 导出的 Excel 包含哪些内容？

包含展会概览，以及关键的 **商品销量统计表**。  
你可以直接把这个表发给社团主催交差（真的很省心）。
